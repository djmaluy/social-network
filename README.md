### Installation:

- **To clone the project**

```
git clone git@github.com:djmaluy/social-network.git
```

- **Go to the folder**

```
cd social-network
```

- **Execute command**

```
npm install
```

### Run the application:

- **For quick application run use this command**

```
npm start
```
